# 🚀 Insurance Fraud Detection and Analysis System

## 📌 Overview

This project is a **Python-based web application built using Django** that focuses on detecting and analyzing fraudulent insurance claims. It leverages **data analysis and machine learning concepts** to identify suspicious patterns in claim data, helping insurance companies reduce financial losses and improve efficiency.

Insurance fraud is a major global issue causing billions in losses annually. This system provides an automated solution to flag potentially fraudulent claims and assist in faster and more accurate decision-making.

---

## 🎯 Objectives

* Detect fraudulent insurance claims using data analysis techniques
* Improve efficiency compared to manual claim verification
* Reduce financial losses for insurance companies
* Enable faster processing of genuine claims
* Provide insights into fraud patterns for future prevention 

---

## 🌍 Scope

* Analyze large datasets of insurance claims
* Identify anomalies and suspicious patterns
* Apply machine learning concepts for fraud detection
* Improve prediction accuracy using data-driven techniques

## 🧠 Methodology

The system follows a structured machine learning workflow:

* **Data Collection:** Gathering insurance claim data
* **Data Preprocessing:** Cleaning and handling missing data
* **Exploratory Data Analysis (EDA):** Identifying patterns and anomalies
* **Clustering & Analysis:** Grouping data to detect unusual behavior
* **Model Development:** Building predictive models for fraud detection

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Yashwanth-2316/insurance-fraud-detection.git
cd insurance-fraud-detection
```

---

### 2️⃣ Create Virtual Environment

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

#### Mac/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Apply Migrations

```bash
python manage.py migrate
```

---

### 5️⃣ Run the Server

```bash
python manage.py runserver
```
